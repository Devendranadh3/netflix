## Project Overview
The **Netflix Content Analysis Dashboard** is a Power BI project that analyzes Netflix movies and TV shows using interactive visualizations.  
The dashboard transforms raw Netflix data into meaningful insights related to content distribution, genres, countries, release trends, ratings, and user-driven exploration.

This project demonstrates practical skills in **data cleaning, data modeling, DAX calculations, and dashboard design** using Microsoft Power BI.

---

## Dataset Information
- **Dataset Name:** Netflix Movies and TV Shows
- **Source:** Kaggle  
  https://www.kaggle.com/datasets/shivamb/netflix-shows
- **Data Type:** CSV
- **Records:** Movies and TV Shows available on Netflix

## Tools & Technologies Used
- **Microsoft Power BI Desktop**
- **Power Query Editor** (Data Cleaning & Transformation)
- **DAX (Data Analysis Expressions)** for measures and KPIs
- **Interactive Visuals & Slicers**
- **Data Modeling** (Handling one-to-many relationships)

---

## Dashboard Objectives & Features

### 1. Overview Analysis
- Total number of titles
- Total Movies and TV Shows
- Content type distribution (Movies vs TV Shows)
- Year-wise content growth trend

### 2. Genre Analysis
- Top genres available on Netflix
- Genre distribution using bar charts and treemaps
- Comparison of genres by content type (Movie / TV Show)

### 3. Country-wise Content Analysis
- Top content-producing countries
- Global distribution using map visuals
- Movies vs TV Shows comparison by country

### 4. Release Year & Rating Analysis
- Content growth trend over release years
- Rating-wise content distribution
- Comparison of ratings between Movies and TV Shows

### 5. Interactivity & User Experience
- Page navigation buttons
- Interactive slicers (Type, Genre, Country, Year, Rating)
- Search-enabled filters
- Dynamic cross-filtering between visuals
- Netflix-inspired dark theme for better UI

---

## Key Insights
- Netflix content has grown rapidly in recent years.
- Movies dominate the platform, but TV Shows are steadily increasing.
- Drama, International content, and Comedy are the most common genres.
- The United States and India are major content producers.
- Ratings such as **TV-MA** and **TV-14** dominate, indicating focus on mature and teenage audiences.

   - Country
   - Release Year
   - Rating
3. Navigate between pages using buttons
4. Interact with visuals to explore insights dynamically
